# compounding Communication

## Specific Over Generic

Bad:

```text
Test more carefully next time.
```

Good:

```text
When a controller adds a new optional query parameter, carry it through the view model and preserve the old paging path so the filter remains non-breaking.
```

## Learnings Summary Template

Each learning should say:

1. what happened
2. why it mattered
3. when to apply it again
4. what future work should do

## Critical Entry Template

Keep promoted entries short:

```text
When <condition>, do <recommended action>. This prevents <repeated failure or wasted effort>. Full entry: history/learnings/<file>.md
```

## Completion Template

```text
Compounding complete for <route>. Learnings file: <path>. Critical promotions: <count>.
```
