# Meta Skills

Meta skills maintain and improve Beer itself rather than running the day-to-day
delivery workflow.

## Skills

| Skill | Purpose |
|---|---|
| `writing-beer-skills` | create or update Beer skills safely |
| `xia` | research external skill repos and map candidates back into Beer |

## Relationship to the Main Workflow

| Category | Main question |
|---|---|
| Workflow | How do we build, debug, and verify this change? |
| Support | What focused utility pass do we need right now? |
| Meta | How do we improve Beer itself? |
